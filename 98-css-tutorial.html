98 CSS Framework Tutorial

If you want to check out the video:



What you need:


  • Code Editor (Like Visual Studio Code)
  • Internet Browser


To start you will need to add the link of 98.css to your .html file


Include 98.css in your webpage -->


Code:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>NAME OF YOUR WEBPAGE</title> <link rel="stylesheet" href="https://unpkg.com/98.css" > </head>
